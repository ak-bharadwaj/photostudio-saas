# PhotoStudio SaaS

A professional Photo Studio Management SaaS platform.

## Features
- Studio Management
- Booking System
- Client Portals
- Image Galleries
- Payment Integration

## Tech Stack
- Frontend: Next.js
- Backend: Node.js / Prisma
- Database: PostgreSQL
- Authentication: Clerk / Firebase

## Getting Started
1. Clone the repository
2. Install dependencies: `pnpm install`
3. Set up environment variables
4. Run migrations: `pnpm prisma generate`
5. Start development: `pnpm dev`
